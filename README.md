# Foundation Calendar

BSc Psychology Foundation Year – Autumn 2025  
Goldsmiths, University of London

## 📚 Available Resources

This repository contains three different views of the Foundation Year Psychology timetable:

1. **[Weekly Calendar](calendar.html)** - Weekly teaching schedule with lectures, tutorials, and seminars organized by day and time
2. **[Full Timetable](timetable.html)** - Complete timetable view showing all teaching weeks in a structured table format
3. **[Week Crosswalk](weekcrosswalk.html)** - Reference guide mapping college weeks to teaching weeks with dates

## 🌐 GitHub Pages

The calendar files are published on GitHub Pages. Once enabled, you can access them at:
- `https://littlemonkeylab.github.io/FoundationCalendar/`
